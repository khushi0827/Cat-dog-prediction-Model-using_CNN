
## Cat–Dog Image Classifier using CNN

### 📌 Project Overview  
This project demonstrates a **Convolutional Neural Network (CNN)** built with **Python, TensorFlow/Keras, and OpenCV** to classify images as either a **cat** or a **dog**. It’s a complete deep learning workflow — from preprocessing to model training, evaluation, and prediction visualization.

---

###  Tech Stack  
- **Python**  
- **TensorFlow/Keras**  
- **OpenCV**  
- **NumPy & Pandas**  
- **Matplotlib/Seaborn** for visualization  

---

###  Features  
- End‑to‑end pipeline: preprocessing, augmentation, CNN architecture, training & evaluation  
- Accuracy tracking with plots of loss/validation curves  
- Prediction samples with visual outputs  
- Easy reproducibility with requirements.txt  

---

### Results  
- Achieved high accuracy on test dataset  
- Visualized predictions with clear cat  vs dog  outputs
